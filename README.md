# QRBox  

ブラウザで QR コードを読み取りたい時は **QRBox** を使いましょう。

## 使い方  

1. サイト上で右クリックして **「QRコードを読み込む」** を選択  
   または、拡張機能のポップアップで **「スキャンする」** をクリック  
2. 読み取りたい範囲をドラッグして選択  
3. 読み取りに成功すると、ポップアップに結果が表示されます  
   - コピー可能  
   - **自動コピーの ON/OFF** も同じ場所で切り替え可能

## 機能  

- 画面上の任意の範囲を選択して QR を解析  
- jsQR を使用した高速デコード  
- 軽量・シンプルな UI  
- 右クリックメニュー対応

## インストール（開発者向け）  

1. このリポジトリを clone  
2. Chrome の `chrome://extensions` を開く  
3. 「デベロッパーモード」を ON  
4. 「パッケージ化されていない拡張機能を読み込む」からこのフォルダを選択
